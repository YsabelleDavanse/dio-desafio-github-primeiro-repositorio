*** Comandos no Windows ***

GIT Bash é um terminal extendido para otimizar o uso do Git

>>>>  Alguns comandos importantes <<<<

- git clone (copiar o link do github)

- git add . ou git add -A (para adicionar todos os arquivos novos)

- git commit -m "escrever mensagem coerente à alteração."

- git push origin main (para enviar os commits para o Github)

--> dir - lista as pastas
--> ctrl+l - limpa o terminal
--> TAB - auto completa
--> kdir - cria nova pastals
--> del - apaga o que está dentro da pasta
--> rmdir - apaga a pasta
